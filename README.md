🚀 DevJourney
📖 Sobre o Projeto

O DevJourney é um site desenvolvido para apresentar minha trajetória durante o curso Técnico em Desenvolvimento de Sistemas. O objetivo é demonstrar a evolução dos conhecimentos adquiridos ao longo do semestre, destacando conteúdos estudados, projetos realizados e metas para o futuro.

O site foi desenvolvido utilizando HTML5 e CSS3, seguindo boas práticas de organização, semântica e responsividade.

🎯 Objetivos
Apresentar minha evolução acadêmica durante o semestre.
Organizar os conteúdos estudados em uma linha do tempo.
Exibir projetos desenvolvidos ao longo do curso.
Demonstrar conhecimentos em desenvolvimento web.
Aplicar conceitos de estruturação semântica e estilização moderna.
